# APCASE

<p text-align='jusfify'>O APCASE é um projeto do laboratório de instrumentação e engenharia biomédica (LIEB) da UFERSA campus Caraúbas. Seu principal objetivo é o estudo do sinal mioelétrico para análise, classificação e predição de suas principais características, com o intuito de desenvolver tecnologias para o campo da engenharia biomédica.
Diante desses processos, existem diversas aplicações associadas à utilização do sinal de eletromiografia. Dentre essas, a confecção de próteses miolétricas inteligentes de baixo custo, que é uma das linhas de pesquisa do projeto APCASE, objetivando facilitar a vida das pessoas.</p>
